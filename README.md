# ArmyChess
ArmyChess 是一款运行在 Android 上的军棋游戏。你可以在悠闲时分与朋友来一局对战。

## 特点
***
 * 支持蓝牙对战
 * 支持自定义阵型

 * 支持悔棋操作

 * 支持认输操作

 * 完整的军棋规则

## 编译环境
---
Android Studio
